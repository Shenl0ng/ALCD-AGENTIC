# Scope and Constraints

## ADLC Binding
Phase: Scope & Constraints.
Control Type: Mandatory operating control.
Block Authority: Yes.

## Operational Control
This project is limited to markdown architecture for a paper-trading-only agent system.

## In Scope
- Agent roles and boundaries
- ADLC controls
- Paper-trading gate architecture
- Memory file contracts
- Evaluation scenarios
- Routine sequencing

## Out of Scope
- Application code
- Broker connections
- Alpaca integration
- API keys or environment files
- Live execution
- Runnable scaffolds
- Automated order placement

## Block Condition
Any request or file that creates execution capability, credentials, or runnable infrastructure is outside scope.
