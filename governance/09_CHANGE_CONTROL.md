# Change Control

## ADLC Binding
Phase: Maintenance & Learning.
Control Type: Mandatory operating control.
Block Authority: Yes.

## Operational Control
No strategy, risk, autonomy, routine, or evaluation rule changes may be made casually.

## Required Change Record
- Change requested
- Reason
- Affected files
- Expected behavior change
- New failure modes
- Human approval
- Review date

## Prohibited Changes Without Approval
- Risk limit increases
- New execution capability
- New agent authority
- Removal of veto gates
- Relaxed no-trade rules

## Block Condition
Unapproved changes to controls invalidate related paper-trade approvals.
