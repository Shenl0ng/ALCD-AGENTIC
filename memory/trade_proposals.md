# Trade Proposals

## ADLC Binding
Phase: Simulation & Proof of Value.

## Purpose
Stores structured paper-trade proposals created only after prior gates pass.

## Required Fields
- Proposal ID
- Symbol
- Direction
- Context
- Liquidity location
- Timing window
- Confirmation
- Entry hypothesis
- Invalidation
- Risk basis
- Paper mode label

## Control Rule
A proposal is not approval and cannot be executed.
