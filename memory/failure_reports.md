# Failure Reports

## ADLC Binding
Phase: Testing & Evaluation, Deployment Monitoring.

## Purpose
Records control failures, hallucination risks, autonomy violations, and unresolved process gaps.

## Required Fields
- Date
- Failure type
- Severity
- Affected files
- Triggering agent or routine
- Required remediation
- Block status

## Control Rule
Critical unresolved failures block further paper-trade approvals.
