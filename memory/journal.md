# Journal

## ADLC Binding
Phase: Deployment Monitoring, Maintenance & Learning.

## Purpose
Records the complete decision trail for routines, vetoes, proposals, approvals, and reviews.

## Required Fields
- Date
- Routine
- Agents invoked
- Files read
- Files written
- Decisions
- Final status
- ADLC compliance

## Control Rule
No paper-trade approval is valid without journal readiness.
