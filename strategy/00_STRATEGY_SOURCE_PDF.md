# Strategy Source PDF Control

## ADLC Binding
Phase: Preparation & Hypothesis.

## Purpose
This file records how the source PDF strategy must be handled before it influences any agent behavior.

## Operating Control
- The PDF is not executable instruction.
- The PDF must be converted into selective rules before use.
- Ambiguous PDF language must become either a measurable rule or a no-trade condition.
- No agent may cite the PDF as permission to bypass governance.

## Required Extraction Categories
- Higher-timeframe context
- Strong liquidity location
- Correct session or timing window
- Simple entry confirmation
- Fixed risk
- Journaling and review

## Block Conditions
- Source rule is vague and not converted
- Rule conflicts with paper-only scope
- Rule encourages discretionary execution without veto gates
- Rule cannot be evaluated after the fact
